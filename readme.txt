Application that plays music for a competitive ballroom final from predefined YouTube Music playlists

Usage:
	python ballroom_final.py [optional flags]

Description:
	
	--klasse, -k: Specify the class (d for D-class, c for C-class, b for B-class and up), default: b
	--length, -l: Specify length of songs (long for 1:50-2:10, normal for 1:30-1:50), default: normal
	--pause, -p: Specify pause time between titles in seconds, default: 30
